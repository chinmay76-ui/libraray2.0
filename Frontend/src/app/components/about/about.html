What is this LMS?

This Library Management System is a web-based application built to automate everyday library tasks such as adding books, managing copies, handling borrow requests, tracking due dates, and sending reminders to students.

The goal is to replace manual registers with a clean digital dashboard so librarians spend less time on paperwork and more time helping readers discover the right books.

Tech Stack

MySQL

Key Features

  • Librarian dashboard to manage books and copies.
  • Student portal with search by title and author.
  • Borrow requests, due dates, and approval flow.
  • Email OTP and reminders via Brevo SMTP.
  • REST APIs with MySQL storage.
  • Fully responsive Angular + Bootstrap 5 UI.
